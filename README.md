# 📚 My Web3 Learning Path

Welcome to my personal journey into the world of Blockchain and Web3 development. This repository contains an interactive roadmap and a collection of resources I've gathered as a student learner.

> **Note:** This is not a professional guide, but rather a "learning log" documenting my progress and sharing what I've found useful so far.

## 🌐 Live Preview
You can view the interactive roadmap here:
👉 **[https://lambertus-daniel.github.io/my-web3-learning-path/](https://lambertus-daniel.github.io/my-web3-learning-path/)**

## ✨ Features
- **Structured Phases:** From Blockchain fundamentals to Advanced Smart Contract Security.
- **Interactive Checklists:** Track progress through various modules.
- **Curated Resources:** Links to documentation, tutorials, and bootcamps (Ethereum, Solidity, Rust, and more).
- **Student Perspective:** Focused on the learning process and building a solid foundation.

## 🛠 Built With
- **HTML5 & Vanilla CSS:** For a clean, lightweight, and interactive UI.
- **JavaScript:** To handle roadmap navigation and progress tracking.
- **GitHub Pages:** For easy and free deployment.
